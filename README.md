## Reqeuriments

1. NextJS
2. NodeJS
3. MaterialUI
4. Firebase

### Frontend

1. Clone this repo

```bash
git clone https://github.com/Rafael-Urei/todo.git
```

2. Go to the project folder

```bash
cd todo
```

3. Install dependencies

```bash
npm install
```

3. Run Frontend

```bash
npm run dev
```
